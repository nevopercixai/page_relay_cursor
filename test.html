PageRelay Extension Test Page

Test Instructions

  1. Install the PageRelay extension
  2. Click the extension icon and set backend URL to: http://localhost:3000/api
  3. Start the test server: node test-server.js
  4. Reload this page
  5. Check the browser console for any errors
  6. Check the test server console to see received data
  7. If injectHtml is working, you should see a green box at the top of this page

Expected Data Collection

The extension should collect:

  • URL: Should match window.location.href
  • Title: "PageRelay Test Page"
  • Meta Description: "This is a test page description for PageRelay extension testing"
  • Selected Text: Empty (unless you select text before reload)
  • HTML Snapshot: Full HTML of this page
